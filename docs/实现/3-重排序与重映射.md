# 脚本 3 设计文档：重排序与重映射 `reorder_entries.py`

---

## 1. 输入参数

```bash
python reorder_entries.py --input <拆分后世界书.json> [--output <输出路径.json>] [--wrapper-name <名称>]
```

| 参数 | 必需 | 默认值 | 说明 |
|---|---|---|---|
| `--input` | 是 | — | 脚本 2 输出的拆分后世界书 JSON |
| `--output` | 否 | `{输入文件名}_reordered.json` | 重排后世界书输出路径 |
| `--wrapper-name` | 否 | `"补充内容"` | 补充包裹条目的内容名称 |

---

## 2. 处理流程

```
输入: 拆分后世界书.json
    │
    ▼
1. load_world_book(input) → entries 列表
    │
    ▼
2. 逐条目判定 is_static（detect_markers）
   保存 _original_order = entry["order"]
    │
    ▼
3. ★ 检测边界条目对 ★
    ├─ 识别开/闭边界条目（未闭合 XML 标签 / "# XXX 开始/结束"）
    ├─ 同 position 内栈式配对
    └─ 对每对边界：
        └─ 检查 order 区间内是否有动态条目
            ├─ 有 → 复制边界副本（仅保留标签），追加到 entries
            └─ 无 → 跳过
    │
    ▼
4. ★ 检测补充包裹风格 ★
    ├─ 统计 XML 标签 vs Markdown 标题使用频率
    ├─ 决定用 <补充内容>...</补充内容> 或 # 补充内容开始...# 补充内容结束
    └─ 若世界书含动态条目 → 创建补充包裹条目，追加到 entries
    │
    ▼
5. ★ 排序 ★
    key = (category, stability, position, depth_key, _original_order)
    category: 0=纯静态, 1=动态/边界副本/补充包裹
    │
    ▼
6. ★ 排序后去空 ★
    ├─ 边界副本对之间无动态条目 → 删除
    └─ 无任何动态条目 → 删除补充包裹
    │
    ▼
7. ★ 重分配 order ★（0, 1, 2, ...）
    │
    ▼
8. ★ 应用字段修改规则 ★
    ├─ 边界副本/补充包裹 → 特殊规则（绿灯 + key=["/.*/"]）
    ├─ 静态条目 → 规则不变
    └─ 动态条目 → 规则不变
    │
    ▼
9. 清理内部临时字段
    _is_static, _original_order, _is_boundary_copy, _is_supplement
    │
    ▼
10. save_world_book → {原名}_reordered.json
```

## 4. 边界条目检测与复制

### 4.1 判定条件

```python
def is_boundary_entry(content: str) -> tuple[bool, str, str, str] | None:
    """
    返回 (is_boundary, type, name, copy_content) 或 None。

    type: "xml_open" | "xml_close" | "md_open" | "md_close"
    name: 标签名或标题基名
    copy_content: 复制时仅保留标签/标题行
    """
```

需同时满足：
1. **无动态标记**（`detect_markers(content)` 为空）
2. **content 存在未闭合的 XML 标签**，或**以 `#`/`##` 开头且以 `开始`/`结束`/`start`/`end` 结尾**

### 4.2 XML 边界检测

```python
def _detect_xml_boundary(content: str) -> tuple | None:
    """
    检测未闭合 XML 开/闭标签。

    <暗部>           → xml_open,  name="暗部", copy="<暗部>"
    <暗部>描述文本    → xml_open,  name="暗部", copy="<暗部>"   (仅标签)
    </暗部>          → xml_close, name="暗部", copy="</暗部>"
    <暗部>...</暗部>  → 不是边界（已闭合）
    """
    stripped = content.strip()

    # 开标签
    m = re.match(r"<([\u4e00-\u9fff\w]+)>\s*", stripped)
    if m:
        tag = m.group(1)
        if f"</{tag}>" not in stripped:
            return ("xml_open", tag, f"<{tag}>")

    # 闭标签
    m = re.match(r"</([\u4e00-\u9fff\w]+)>", stripped)
    if m:
        tag = m.group(1)
        return ("xml_close", tag, f"</{tag}>")

    return None
```

### 4.3 Markdown 边界检测

```python
def _detect_md_boundary(content: str) -> tuple | None:
    """
    检测 Markdown 标题分界标记。

    # 暗部开始          → md_open,  name="暗部", copy="# 暗部开始"
    ## 暗部结束         → md_close, name="暗部", copy="## 暗部结束"
    # 暗部开始\n描述文本  → md_open,  name="暗部", copy="# 暗部开始"  (仅标题)
    """
    stripped = content.strip()
    lines = stripped.splitlines()

    for line in lines:
        m = re.match(
            r"^(#{1,2})\s+(.+?)(开始|结束|start|end|begin|close)\s*$",
            line,
        )
        if m:
            heading_level = m.group(1)
            base_name = m.group(2).strip()
            suffix = m.group(3)
            btype = "md_open" if suffix in ("开始", "start", "begin") else "md_close"
            copy_content = f"{heading_level} {base_name}{suffix}"
            return (btype, base_name, copy_content)

    return None
```

### 4.4 配对算法

```python
def pair_boundaries(entries: list[dict]) -> list[BoundaryPair]:
    """
    同 position 内栈式配对。

    BoundaryPair:
        open_entry: dict      # 开边界原始条目
        close_entry: dict     # 闭边界原始条目
        open_copy_content: str
        close_copy_content: str

    规则：
    - 同 tag_name/base_name + 同 position
    - 栈式最近邻配对
    """
    pairs = []
    groups = defaultdict(list)
    for e in entries:
        groups[e["position"]].append(e)

    for pos, group in groups.items():
        stack = []
        for e in sorted(group, key=lambda x: x["_original_order"]):
            r = is_boundary_entry(e["content"])
            if r is None:
                continue
            btype, name, copy_content = r
            if btype in ("xml_open", "md_open"):
                stack.append((e, name, copy_content))
            else:  # xml_close, md_close
                for i in range(len(stack) - 1, -1, -1):
                    if stack[i][1] == name:
                        open_e, _, open_copy = stack.pop(i)
                        pairs.append(
                            BoundaryPair(
                                open_entry=open_e,
                                close_entry=e,
                                open_copy_content=open_copy,
                                close_copy_content=copy_content,
                            )
                        )
                        break
    return pairs
```

### 4.5 边界副本创建

```python
def create_boundary_copies(
    entries: list[dict], pairs: list[BoundaryPair]
) -> list[dict]:
    """
    对每对边界，检查 order 区间内是否有动态条目。
    有 → 创建副本（开+闭），追加到 entries。
    无 → 不创建。
    """
    new_copies = []

    for pair in pairs:
        min_order = pair.open_entry["_original_order"]
        max_order = pair.close_entry["_original_order"]

        has_dynamic = any(
            min_order < e["_original_order"] < max_order
            and not determine_static(e["content"])
            for e in entries
        )

        if not has_dynamic:
            continue

        # 开放边界副本
        open_copy = deepcopy(pair.open_entry)
        open_copy["content"] = pair.open_copy_content
        open_copy["_is_boundary_copy"] = True
        open_copy["_original_order"] = min_order
        open_copy["comment"] = (
            f"{pair.open_entry.get('comment', '')} [boundary-copy-open]"
        ).strip()
        open_copy["uid"] = None

        # 闭合边界副本
        close_copy = deepcopy(pair.close_entry)
        close_copy["content"] = pair.close_copy_content
        close_copy["_is_boundary_copy"] = True
        close_copy["_original_order"] = max_order
        close_copy["comment"] = (
            f"{pair.close_entry.get('comment', '')} [boundary-copy-close]"
        ).strip()
        close_copy["uid"] = None

        new_copies.extend([open_copy, close_copy])

    return new_copies
```

## 5. 补充包裹条目生成

### 5.1 风格检测

```python
def detect_wrapper_style(entries: list[dict]) -> str:
    """
    统计 XML 标签 vs Markdown 标题出现频率。
    返回 "xml" 或 "markdown"。
    """
    xml = 0
    md = 0
    for e in entries:
        xml += len(re.findall(r"<[\u4e00-\u9fff\w]+>", e["content"]))
        md += len(re.findall(r"^#{1,2}\s", e["content"], re.MULTILINE))
    return "xml" if xml >= md else "markdown"
```

### 5.2 包裹内容

| 风格 | 开放内容 | 闭合内容 |
|---|---|---|
| XML | `<补充内容>` | `</补充内容>` |
| Markdown | `# 补充内容开始` | `# 补充内容结束` |

用户可通过 `--wrapper-name` 覆盖名称：`--wrapper-name "可变内容"` → `<可变内容>` / `# 可变内容开始`。

### 5.3 创建包裹条目

```python
def create_supplement_wrapper(
    entries: list[dict], wrapper_name: str
) -> list[dict]:
    """
    若世界书含动态条目，创建补充包裹开+闭条目。
    否则返回空列表。
    """
    has_dynamic = any(not determine_static(e["content"]) for e in entries)
    if not has_dynamic:
        return []

    style = detect_wrapper_style(entries)

    if style == "xml":
        open_content = f"<{wrapper_name}>"
        close_content = f"</{wrapper_name}>"
    else:
        open_content = f"# {wrapper_name}开始"
        close_content = f"# {wrapper_name}结束"

    open_entry = {
        "content": open_content,
        "comment": "[supplement-start]",
        "constant": False,
        "key": [],
        "keysecondary": [],
        "position": 4,
        "depth": 0,
        "role": 1,
        "order": 0,
        "disable": False,
        "selective": True,
        "selectiveLogic": 0,
        "probability": 100,
        "useProbability": True,
        "sticky": 0,
        "cooldown": 0,
        "delay": 0,
        "uid": None,
        "_is_supplement": True,
        "_original_order": float("inf"),  # 确保排在动态条目区域末尾
    }

    close_entry = deepcopy(open_entry)
    close_entry["content"] = close_content
    close_entry["comment"] = "[supplement-end]"

    return [open_entry, close_entry]
```

## 6. 排序

### 6.1 排序键

```python
def sort_key(entry: dict) -> tuple:
    is_static = determine_static(entry["content"])
    is_copy_or_supplement = (
        entry.get("_is_boundary_copy")
        or entry.get("_is_supplement")
    )

    # category: 纯静态=0, 动态/边界副本/补充包裹=1
    category = 1 if (not is_static or is_copy_or_supplement) else 0

    # sub_order: 补充包裹在外层
    if entry.get("_is_supplement"):
        comment = entry.get("comment", "")
        if "[supplement-start]" in comment:
            sub_order = 0   # 开包裹: category=1 组内第1位
        else:
            sub_order = 2   # 闭包裹: category=1 组内最后
    else:
        sub_order = 1       # 边界副本 + 动态条目: 中间

    pos = entry["position"]
    depth = entry["depth"]

    if pos == 4:
        if depth >= 10:
            stability = 0
        else:
            stability = 2
        depth_key = -depth
    else:
        stability = 1
        depth_key = 0

    return (
        category,
        sub_order,         # 0=包裹开, 1=内容, 2=包裹闭
        stability,
        pos,
        depth_key,
        entry["_original_order"],
    )
```

> 边界副本的 `_original_order` 设为关联动态条目的最小/最大 order，确保开副本在动态首、闭副本在动态尾。
>
> 补充包裹的 `_original_order` 设为 `inf`，确保排到动态区域末尾。

## 7. 排序后去空

```python
def prune_empty_wrappers(entries: list[dict]) -> list[dict]:
    """
    1. 边界副本对之间无动态条目 → 删除这对副本
    2. 无任何动态条目 → 删除所有补充包裹条目
    """
    # 收集边界副本索引
    boundary_indices = []
    for i, e in enumerate(entries):
        if e.get("_is_boundary_copy"):
            boundary_indices.append(i)

    to_remove = set()

    # 配对并检查
    i = 0
    while i < len(boundary_indices):
        open_idx = boundary_indices[i]
        close_idx = boundary_indices[i + 1] if i + 1 < len(boundary_indices) else None
        if close_idx is None:
            break

        has_dynamic = any(
            not determine_static(entries[j]["content"])
            and not entries[j].get("_is_boundary_copy")
            and not entries[j].get("_is_supplement")
            for j in range(open_idx + 1, close_idx)
        )

        if not has_dynamic:
            to_remove.update([open_idx, close_idx])

        i += 2

    # 检查补充包裹
    has_any_dynamic = any(
        not determine_static(e["content"])
        and not e.get("_is_boundary_copy")
        for e in entries
    )
    if not has_any_dynamic:
        for i, e in enumerate(entries):
            if e.get("_is_supplement"):
                to_remove.add(i)

    return [e for i, e in enumerate(entries) if i not in to_remove]
```

## 8. 字段修改规则

### 8.1 规则表

```python
def apply_rules(entry: dict) -> None:
    is_static = entry["_is_static"]
    is_copy = entry.get("_is_boundary_copy")
    is_supplement = entry.get("_is_supplement")

    if is_copy or is_supplement:
        # 边界副本 / 补充包裹 → 绿灯 + 通配符
        entry["constant"] = False
        entry["position"] = 4
        entry["depth"] = 0
        entry["role"] = 1
        entry["key"] = ["/.*/"]
        entry["useProbability"] = False
        entry["probability"] = 100
        entry["cooldown"] = 0
        entry["sticky"] = 0
        entry["delay"] = 0
        return

    pos = entry["position"]
    depth = entry["depth"]

    if is_static:
        entry["constant"] = True
        entry["cooldown"] = 0
        entry["probability"] = 100
        entry["useProbability"] = False
        entry["sticky"] = 0
        entry["delay"] = 0

        if pos == 4:
            if depth < 10:
                entry["depth"] = 0
            else:
                entry["depth"] = 9999
            entry["role"] = 1
        # else: position/depth/role 保持原值
    else:
        entry["position"] = 4
        entry["depth"] = 0
        entry["role"] = 1
        # constant/cooldown/probability/useProbability/sticky/delay 保持原值
```

### 8.2 字段速查表

| 字段 | 纯静态 pos4 <10 | 纯静态 pos4 >=10 | 纯静态 pos!=4 | 纯动态 | 边界副本/补充包裹 |
|---|---|---|---|---|---|
| `position` | 4 | 4 | **不变** | 4 | 4 |
| `depth` | 0 | 9999 | **不变** | 0 | 0 |
| `role` | 1 | 1 | **不变** | 1 | 1 |
| `constant` | true | true | true | **不变** | false |
| `cooldown` | 0 | 0 | 0 | **不变** | 0 |
| `probability` | 100 | 100 | 100 | **不变** | 100 |
| `useProbability` | false | false | false | **不变** | false |
| `sticky` | 0 | 0 | 0 | **不变** | 0 |
| `delay` | 0 | 0 | 0 | **不变** | 0 |
| `key` | **不变** | **不变** | **不变** | **不变** | `["/.*/"]` |
