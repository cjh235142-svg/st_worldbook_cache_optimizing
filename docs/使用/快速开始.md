# 快速开始

## 环境要求

- Python 3.10+
- （可选）pytest — 运行测试：`pip install pytest`

## 一键优化

```bash
python -m scripts.run_pipeline -i 学园都市0313.json
```

或者指定输出目录：

```bash
python -m scripts.run_pipeline -i 学园都市0313.json -d result
```

## 产物链

对 `学园都市0313.json` 运行管线后生成：

```
学园都市0313.backup_20260602_005135.json    ← 原始备份（安全网）
学园都市0313_analysis.json                  ← 脚本1：分析结果
学园都市0313_split.json                     ← 脚本2：拆分后
学园都市0313_reordered.json                 ← 脚本3：重排序后
学园都市0313_merged.json                    ← 脚本4：最终产物 ★
```

`_merged.json` 是**最终优化产物**，可直接导入酒馆使用。

## 运行测试

```bash
# 全部测试（142 个）
pytest scripts/tests/ -v

# 仅集成测试
pytest scripts/tests/test_integration.py -v

# 仅单元测试
pytest scripts/tests/ --ignore=scripts/tests/test_integration.py -v
```

## 在酒馆中使用优化产物

1. 打开 SillyTavern，进入 World Info 管理页
2. 导入 `_merged.json` 为新的世界书
3. 在角色卡设置中关联该世界书
4. 配置 DeepSeek API 即可

## 补充包裹名称

动态内容默认使用 `<补充内容>...</补充内容>` 包裹。可自定义：

```bash
python -m scripts.run_pipeline -i 世界书.json -w "可变内容"
```

输出：`<可变内容>...</可变内容>`（XML 风格）或 `# 可变内容开始...# 可变内容结束`（Markdown 风格）。
